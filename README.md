# Malaysian Sign Language

This repository will be used for training and deploying the Malaysian sign language, as part of WQF7006 Computer Vision and Image Processing assignment.

## Prerequisites

Install `pixi` from [Homebrew](https://formulae.brew.sh/formula/pixi).

## Setup

1) Clone this repository.
2) `git lfs install`
3) `git lfs track "*.pt" && git lfs track "*.npy"`

## Run Website

`pixi run server`
